AlgaT è un software scritto in Java con la libreria JavaFX per l'autoapprendimento interattivo della struttura dati Heap e dell'algoritmo Heapsort. 

E' possibile inserire nuove domande tramite i file presenti nella cartella "questions_NOME_LEZIONE.txt".
Per aggiungere nuove domande basta porre domande e risposte seguite dalla barra verticale "|" e dal punto fermo "." rispettivamente come mostrato di seguito:

Domanda 1|Risposta 1.
Domanda 2|Risposta 2.

Domande e risposte sono considerate indipendentemente da spazi bianchi e "case".

Il file .jar deve trovarsi nella stessa cartella degli altri file per funzionare.

Il codice è stato scritto interamente da Fogu Gabriele e Mylonopoulos Dario, è open source e non è stato realizzato a fini commerciali.
